# Проект обработки банковских операций

## Описание
Учебный Python-проект для обработки информации о банковских картах, счетах и операциях.
Проект реализует маскировку данных, фильтрацию и сортировку операций.

---

## Функциональность

Проект содержит следующие функции:

### Маскировка данных
- `get_mask_card_number` — маскировка номера банковской карты
- `get_mask_account` — маскировка номера банковского счета
- `mask_account_card` — обработка строки с типом и номером карты или счета

### Обработка операций
- `filter_by_state` — фильтрация операций по статусу
- `sort_by_date` — сортировка операций по дате

### Работа с датами
- `get_date` — преобразование даты в формат `ДД.ММ.ГГГГ`
